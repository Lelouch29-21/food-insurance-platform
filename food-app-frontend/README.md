# Food App Frontend

React + Vite SPA deployed on GitHub Pages.

## Setup
```bash
cp .env.example .env
npm install
npm run dev
```

## Routes
- `/`
- `/menu`
- `/cart`
- `/orders`
- `/login`
- `/register`

## Features
- Menu fetch (`GET /food`)
- Cart persisted in localStorage
- Checkout (`POST /orders`)
- Interest adjustment response shown after order
